# Call-Center-Performance-Tracker-Excel
📌 Project Overview
This Excel dashboard analyzes call center operations data across 4 US cities — Baltimore/MD, Chicago/IL, Denver/CO, and Los Angeles/CA — covering customer sentiment, SLA compliance, channel performance, and agent CSAT scores. The goal was to identify service quality gaps and improve operational efficiency.

📊 Key Metrics Tracked
Metric                    Value
Total Calls Analyzed      70
Average CSAT Score        5.33
Average Call Duration     26.26 min
sAverage Response Time    70
Channels Tracked          4 (Call Center, Chatbot, Email, Web)

🔍 Key Insights from the Dashboard

Top Call Reason: Billing Questions dominate with 50+ calls — far exceeding Service Outage and Payment queries, suggesting billing process needs simplification
Sentiment Breakdown: Positive and Very Positive sentiments form a significant portion — but Negative and Very Negative calls still present, indicating room for improvement in issue resolution
SLA Compliance Risk: Response time chart shows sharp spike for SLA-breached calls — calls that miss SLA have 3-4x higher average response time than compliant calls
Channel Performance: Call Center and Chatbot handle the most volume — Email channel significantly underutilized, web channel is mid-range
CSAT vs Call Volume: Agent-wise analysis shows some agents handle high call volume but score lower CSAT — training opportunity identified
City Performance: Los Angeles and Chicago show higher call concentration — resource allocation should prioritize these locations


🛠️ Tools & Techniques Used

Tool: Microsoft Excel
Visuals Used: Pie chart, line chart, bar chart, combo chart (bar + line)
Techniques: Pivot Tables, Pivot Charts, Slicers for sentiment and city filtering
Metrics Calculated: Average CSAT, Average Response Time, SLA compliance rate


💼 Business Use Case
This dashboard is built for call center managers and CX teams to:

Track agent performance and CSAT scores for coaching decisions
Identify top reasons for customer calls to reduce inbound volume
Monitor SLA compliance and flag at-risk interactions
Compare channel efficiency to optimize staffing across touchpoints
